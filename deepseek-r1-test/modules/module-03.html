主流方法对比

State-of-the-Art Comparison

当前最强推理模型采用了三种不同的技术路线。让我们横向对比。

三条技术路线

🛠️
路线一：SFT + RLHF + CoT（Claude / GPT-4o）

用海量标注数据训练，CoT prompting 辅助推理。强但需要大量人工标注，推理链质量不可控。

⏱️
路线二：Extended Thinking（OpenAI o1/o3）

在推理时引入"思考 token"，让模型可以花更多算力思考。推理能力大幅提升，但仍是黑箱。

🧠
路线三：Pure RL + 涌现（DeepSeek-R1 / R1-Zero）

无需标注数据，通过 RL 和正确的奖励信号让推理能力自己涌现。可解释性强（思维链可见），可复现性强。

各方法优劣分析

🎯
SFT + RLHF（GPT-4o / Claude）

优点：通用能力强，指令遵循好
缺点：推理专项能力弱，训练成本极高（需要海量人工标注）

⏱️
Extended Thinking（o1 / o3）

优点：推理能力大幅提升，benchmark 表现好
缺点：黑箱不可解释，推理成本高（每次推理消耗大量 tokens），闭源无法研究

🧠
Pure RL（DeepSeek-R1 / R1-Zero）

优点：推理链完全可见可分析，无需标注数据，开源可复现，涌现能力令人惊喜
缺点：R1-Zero 输出可读性差（用语言混合思考），训练稳定性需要技巧
